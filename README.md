## 💻 About Food Ordering Website

- Built using **React.js with Vite** for fast performance  
- Follows **component-based architecture**  
- Designed as a **restaurant/food ordering UI**  
- Includes **menu display, categories, and interactive sections**  
- Focused on **clean design and responsive layout**  
- Highlights **real-world frontend structure and UI patterns**  

### 🌟 Key Features

- **Add/Remove Items:** Easily add food items to the cart or remove them  
- **Quantity Adjustment:** Increase or decrease the number of each item in the cart  
- **Dynamic Total Calculation:** Cart total updates automatically as items are added or removed  
- **Responsive Design:** Works smoothly on desktop, tablet, and mobile screens  
- **Interactive Food Cards:** Each food item is displayed as a card with image, name, description, price, and action buttons  
- **React Component-Based Structure:** Clean, reusable components for scalability and maintenance
